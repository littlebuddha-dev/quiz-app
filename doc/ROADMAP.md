# **🚀 Cue 開発ロードマップ**

「小学生向け論理的思考クイズ」をテーマにした、YouTubeライクな学習プラットフォームの構築を目指します。多言語対応（i18n）を前提とし、子供が自発的に学びたくなるようなUI/UXを提供します。

## **🎯 ビジョン**

「見る」だけでなく「考える」「答える」を促す、次世代の教育エンターテインメント・プラットフォーム。

## Phase 4: 高度な機能とUXの追求

ユーザー体験を劇的に向上させるための 3 つの高度な機能を実装しました。

### 1. ダークモード & プレミアム UI デザイン
システム設定に連動するダークモードを導入し、デザイン全体をよりプレミアム（高級感のある）なものに刷新しました。

- **globals.css**: Tailwind v4 の方式に則り、CSS 変数（`--background`, `--foreground`, `--card`, `--border`）による動的なテーマ構築を実現。
- **UI コンポーネント**: 全てのハードコードされた色を CSS 変数に置き換え、ダークモード時でも視認性が高く、美しいデザインを維持します。

### 2. ゲーミフィケーション (XP & Level システム)
モチベーション維持のため、学習進捗を可視化する仕組みを導入しました。

- **DB 拡張**: `User` モデルに `xp` と `level` を追加。
- **XP 獲得ロジック**: クイズ正解時に +10 XP を獲得。一定量（level * 100）貯まるとレベルアップします。
- **UI 表示**: ヘッダーに現在のレベルと XP プログレスバーを表示。

### 3. AI レコメンド (関連クイズの提案)
視聴中のクイズに基づき、次の学習を提案する機能を実装しました。

- **watch ページ**: 現在のクイズと同じカテゴリ、または近い対象年齢のクイズをサーバーサイドで動的に抽出。
- **おすすめサイドバー**: モックデータを排除し、DB から取得したリアルな関連クイズを表示します。

## **🗺️ 開発フェーズ**

### **Phase 1: データベースと永続化の基盤構築 (Core Backend)**

現在のモックデータをデータベースに移行し、AIが生成したクイズを保存・管理できるようにします。

* **\[ \] データベース構築 (Prisma \+ Supabase PostgreSQL)**  
  * Quiz, User, Category などのテーブル設計。  
  * 多言語対応を見据え、タイトルや問題文の翻訳データを持てるスキーマ設計。  
* **\[ \] クイズのキャッシュ・保存機能**  
  * AI生成時にDBへ保存し、次回以降はDBから高速に読み込む（キャッシュ機能）。  
* **\[ \] YouTube風グリッドUIの動的化**  
  * DBからクイズ一覧を取得し、最新順・人気順でソートして表示。  
  * 無限スクロール（またはページネーション）の実装。  
* **\[ \] 問題の検索・絞り込み機能の高度化**  
  * DBクエリを利用した高速なテキスト検索。

### **Phase 2: ユーザー認証とパーソナライズ (User Experience)**

ClerkまたはBetter Authを利用してユーザー管理を導入し、学習体験を個人に最適化します。

* **\[ \] アカウント機能 (認証)**  
  * Google連携ログイン、保護者用・子供用アカウントの分離。  
* **\[ \] 年齢に合わせた問題表示**  
  * ユーザー登録時に入力した「生年月日（学年）」から、最適な難易度のクイズを初期画面に表示。  
* **\[ \] 学習記録（視聴・回答履歴）**  
  * YouTubeの「履歴」のように、過去に解いた問題と正誤を保存。  
* **\[ \] 「後で解く（お気に入り）」機能**  
  * YouTubeの「後で見る」に相当。お気に入りリストの作成。  
* **\[ \] いいね！・高評価機能**  
  * 問題に対する評価システム。人気順ソートの基準にする。

### **Phase 3: クリエイター機能と管理者機能 (Creator & Admin)**

プラットフォームとしての運用を可能にする管理機能を追加します。

* **\[ \] 管理者ダッシュボード (CMS)**  
  * AIが生成した問題の目視確認、手動での修正・削除機能。  
  * 新しい問題の手動作成フォーム。  
* **\[ \] チャンネル機能（先生・親クリエイター）**  
  * 特定の先生や親が作成した問題集を「チャンネル」としてまとめる機能。  
  * チャンネル登録（フォロー）機能。  
* **\[ \] コメント・ディスカッション欄**  
  * 「なんでこの答えになるの？」を議論できる、モデレーション付きのコメント欄。

### **Phase 4: 高度な機能とUXの追求 (Advanced Features)**

サイトの利便性を最高レベルまで引き上げます。

* **\[ \] AI レコメンドエンジン (おすすめ機能)**  
  * 「関連動画」のように、ユーザーの回答傾向や正答率に基づいて、次に解くべき問題をサイドバーに提案。  
* **\[ \] ゲーミフィケーション要素**  
  * 連続正解でのバッジ獲得、経験値（XP）システム、レベルアップ演出。  
* **\[ \] 完璧な多言語対応 (i18n)**  
  * URLルーティング（例: /ja/quiz/..., /en/quiz/...）によるSEOを意識した多言語化。  
* **\[ \] PWA (Progressive Web App) 対応**  
  * スマホのホーム画面に追加でき、アプリのように動作する機能。オフライン時のキャッシュ対応。  
* **\[ \] ダークモード対応**  
  * 目への負担を減らすため、ユーザーのシステム設定に合わせたテーマ切り替え。

## **🎨 UI/UX 詳細仕様メモ (YouTubeリファレンス)**

* **サムネイル**: 文字入り画像を大きく見せる。右下に「対象年齢」や「クイズのタイプ（選択/記述）」のバッジを表示。
